Su GitHub sono presenti tutti i progetti e le applicazioni che ho sviluppato sin dall'inizio della mia carriera da software developer. La mia esperienza ha avuto inizio con la creazione di Progressive Web Application (PWA) utilizzando i linguaggi Javascript e Typescript insieme a Node.js, React.js e Express.js.
Successivamente ho ampliato il mio campo di competenza nel settore dello sviluppo Android utilizzando il suo linguaggio nativo Kotlin. All'inizio, ho creato applicazioni con una UI basata sugli XML Layouts. Attualmente, ho adottato il toolkit Jetpack Compose, che utilizzo per lo sviluppo delle mie applicazioni Android.
Durante la mia esperienza nella creazione di applicazioni Android, ho utilizzato diverse tecnologie per definire l'interfaccia utente delle mie applicazioni. In particolare, ho sviluppato alcune applicazioni utilizzando il linguaggio XML per definire le viste e i layout dell'interfaccia utente, mentre per altre ho utilizzato Jetpack Compose, il nuovo framework di Android per la creazione di UI dichiarative.
